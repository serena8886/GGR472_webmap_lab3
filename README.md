# GGR472 Lab 3 – Toronto DineSafe Web Map
This web map visualizes restaurant inspection results across Toronto using data from the City of Toronto's DineSafe program. Each restaurant is displayed as a coloured circle based on its inspection status, and users can filter, hover, and click to explore the data.
- Dataset:DineSafe – City of Toronto restaurant inspection results
- Source: [Toronto Open Data](https://open.toronto.ca/dataset/dinesafe/)
